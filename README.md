# Scenario Form

Chrome extension for capturing and replaying reusable form-filling scenarios on complex web forms.

## Overview

Scenario Form helps users capture form values, save them as reusable scenarios, and replay them later in one click.

It is designed for repeated form filling, testing workflows, internal tools, long questionnaires, and dynamic web applications where standard browser autofill is not enough.

## Key Features

- Capture filled form values from a web page
- Replay saved scenarios later
- Manage multiple form scenarios
- Configure custom field detection rules
- Use presets for complex form platforms
- Support nested and custom form controls
- Export and import scenarios and rules
- Store all data locally in the browser

## Privacy

Scenario Form is local-first.

The extension does not send form values, scenario values, page content, cookies, passwords, or personal data to external servers.

All scenarios and rules are stored locally in the user's browser unless the user manually exports them.
