# Privacy Policy

Last updated: 2026-06-25

Scenario Form is a Chrome extension for capturing and replaying reusable form-filling scenarios.

## Data Storage

Scenario Form stores data locally in the user's browser using Chrome extension storage.

The extension may store:

- saved form scenarios;
- form field values saved by the user;
- custom field rules;
- extension settings;
- imported data, if the user manually imports a backup file.

## Data Transmission

Scenario Form does not send form values, scenario values, page content, cookies, passwords, or personal data to external servers.

The current version of Scenario Form does not use remote analytics, tracking pixels, advertising SDKs, or external telemetry services.

## Manual Export and Import

Users can manually export scenarios and rules to a local file.

Exported files are created only after a user action. Users are responsible for storing and sharing exported files securely.

## Bug Reports

Bug reports are submitted manually by the user through GitHub Issues or another support channel.

Users should not include passwords, personal data, private form values, cookies, tokens, screenshots with sensitive information, or private page content in bug reports.

## Permissions

Scenario Form requires browser extension permissions to detect and fill form fields on web pages.

These permissions are used only to provide the extension's core functionality:

- capturing form values;
- replaying saved scenarios;
- detecting supported form fields;
- applying user-configured field rules.

## Closed Source Status

The Scenario Form extension source code is currently closed source.

This public repository is used for product information, support, bug reports, feature requests, privacy policy, and public changelog.

## Contact

For support or privacy-related questions, please use the public support channel linked in this repository.
