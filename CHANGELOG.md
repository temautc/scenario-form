# Changelog

All notable changes to Scenario Form will be documented in this file.

## [0.1.0] - Unreleased

### Added

* Initial MVP version of Scenario Form.
* Form scenario capture.
* Saved scenario replay.
* Scenario management.
* Custom field detection rules.
* Built-in presets for complex form platforms.
* Nested elements support for custom form controls.
* Automatic field highlighting for detected and filled form fields.
* Import and export for scenarios and rules.
* Local browser storage.
* Settings page.
* Privacy-focused support and bug reporting flow.

### Notes

Scenario Form is currently in MVP stage.

The extension source code is currently closed source. This public repository is used for product information, support, bug reports, feature requests, privacy policy, and public changelog.
